# Windows companion (planned)

Windows support is planned for the next milestone. The intended implementation is a tray application with an optional floating panel, using the same read-only Codex app-server methods as the macOS companion:

- `account/rateLimits/read`
- `account/usage/read`

The v0.1.0 release does not ship a Windows executable. Do not download untrusted third-party binaries claiming to be an official Windows build of this project.
